# Paso 4: Modelado con Machine Learning

## 🎯 Objetivo

Entrenar y evaluar modelos de machine learning para predecir enfermedades cardíacas.

## 📋 Tareas

1. Dividir datos en train/test
2. Entrenar múltiples modelos
3. Evaluar rendimiento
4. Seleccionar mejor modelo
5. Guardar modelo final

## 🚀 Instrucciones

```bash
python paso_04_modelado/entrenar_modelo.py
python paso_04_modelado/evaluar_modelo.py
```

## 📊 Resultados Esperados

- `modelo_final.pkl`: Modelo entrenado
- `metricas.txt`: Métricas de evaluación
- `curva_roc.png`: Curva ROC del modelo
