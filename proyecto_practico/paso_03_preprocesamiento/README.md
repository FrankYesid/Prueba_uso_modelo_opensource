# Paso 3: Preprocesamiento de Datos

## 🎯 Objetivo

Preparar y limpiar los datos para el modelado, incluyendo manejo de valores faltantes, codificación de variables y feature engineering.

## 📋 Tareas

1. Limpiar datos
2. Manejar valores faltantes (si existen)
3. Codificar variables categóricas
4. Crear nuevas features
5. Estandarizar variables numéricas
6. Guardar dataset procesado

## 🚀 Instrucciones

### 1. Limpiar Datos

```bash
python paso_03_preprocesamiento/limpieza_datos.py
```

### 2. Feature Engineering

```bash
python paso_03_preprocesamiento/feature_engineering.py
```

## 📊 Resultados Esperados

- `datos_limpios.csv`: Dataset limpio
- `datos_procesados.csv`: Dataset listo para modelado
- `preprocesador.pkl`: Objeto preprocesador guardado
