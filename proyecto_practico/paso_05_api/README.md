# Paso 5: API REST

## 🎯 Objetivo

Crear una API REST para servir predicciones del modelo entrenado.

## 🚀 Instrucciones

```bash
python paso_05_api/api.py
```

Luego en otra terminal:
```bash
python paso_05_api/test_api.py
```

## 📊 Endpoints

- `GET /health`: Health check
- `POST /predict`: Realizar predicción
