# Notebooks de Análisis de Datos

## 📚 Descripción

Esta carpeta contiene notebooks de Jupyter para análisis profundo del dataset CVD_cleaned.csv, desde exploración básica hasta modelado avanzado.

## 📓 Notebooks Disponibles

### 1. [01_exploracion_datos.ipynb](01_exploracion_datos.ipynb)
Análisis exploratorio completo del dataset:
- Carga y estructura de datos
- Estadísticas descriptivas
- Distribuciones de variables
- Valores faltantes y outliers
- Primeras visualizaciones

### 2. [02_analisis_estadistico.ipynb](02_analisis_estadistico.ipynb)
Análisis estadístico avanzado:
- Tests de hipótesis
- Análisis de correlaciones
- Comparaciones entre grupos
- Significancia estadística

### 3. [03_visualizacion_datos.ipynb](03_visualizacion_datos.ipynb)
Visualizaciones avanzadas:
- Gráficos interactivos
- Dashboards
- Análisis de relaciones
- Distribuciones comparativas

### 4. [04_modelos_prediccion.ipynb](04_modelos_prediccion.ipynb)
Modelado con Machine Learning:
- Preprocesamiento
- Entrenamiento de modelos
- Evaluación y comparación
- Optimización de hiperparámetros

## 🚀 Cómo Usar

### Instalación

```bash
# Instalar Jupyter
pip install jupyter notebook

# O usar JupyterLab
pip install jupyterlab
```

### Ejecutar

```bash
# Iniciar Jupyter Notebook
jupyter notebook

# O JupyterLab
jupyter lab
```

### Navegación

1. Abre el notebook que deseas ejecutar
2. Ejecuta las celdas en orden (Shift + Enter)
3. Revisa los resultados y gráficos
4. Experimenta modificando parámetros

## 📊 Requisitos

- Python 3.8+
- Jupyter Notebook o JupyterLab
- Dependencias del proyecto (ver requirements.txt)
